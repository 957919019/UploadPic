### 一个演示了多图上传的Demo
--------------
由项目中剥离出来，演示了如何完成多图上传的逻辑以及UI的控制，在Demo中，你可以学习到
> * 谷歌MVP的使用
> * 简单的BaseActivity的抽离
> * 使用Recyclerview完成选取图片后的展示
> * 使用第三方控件完成UI的效果以及手势操作
